AprendendoPython

Este repositório contém uma coleção de exercícios em Python que aprendi com a plataforma Alura, muito bom para praticar lógica de programação, estruturas de dados e conceitos fundamentais da linguagem. 

Estrutura do repositório

Os arquivos são exercícios basicos com uso de:

Sintaxe, variáveis, operadores, entrada/saída

estruturas | Condicionais, loops, listas

funcoes | Definição e uso de funções

Como executar os exercícios

Certifique-se de ter o Python instalado em seu computador. 

Para rodar um exercício:

python nome_do_arquivo.py

Objetivo

Compartilhar conhecimento adiquirido ao longo do curso, mostrando minhas logica no processo.
